# sql-gpt-bootcamp

## 📌 About

This repository contains my SQL learning journey from scratch, guided by ChatGPT as my personal SQL mentor.

The objective is to build a strong foundation in SQL by understanding concepts deeply, practicing extensively, and solving real-world business scenarios rather than memorizing syntax.

Every topic includes:
- Concept learning
- Practice exercises
- Assignments
- Interview-style questions
- Business scenarios
- Mini projects

---

## 🎯 Objectives

- Master SQL from Beginner to Advanced
- Build strong problem-solving skills
- Prepare for Data Analyst and Data Science interviews
- Learn SQL using industry best practices
- Develop real-world database projects
